**Part A: Create `components/FAQSection.tsx`**

This MUST be a client component (`"use client"` at the top) because it uses `useState` for accordion state and `AnimatePresence` for animation.

Import: `useState` from "react", `motion` and `AnimatePresence` from "motion/react", `faqItems` from "@/lib/content".

**State:** `const [openIndex, setOpenIndex] = useState<number | null>(null)` -- single-open accordion pattern. Clicking an open item closes it (sets null), clicking a closed item opens it and closes the previously open one.

Toggle function: `const toggle = (index: number) => { setOpenIndex(openIndex === index ? null : index) }`

**Section structure:**
- `<section id="faq" className="py-24 px-6">`
- Container: `max-w-3xl mx-auto`
- Heading: `<h2 className="text-3xl md:text-4xl font-bold text-center mb-12">Frequently Asked Questions</h2>`

**FAQ items:** `<div className="space-y-2">` mapping over `faqItems` with index.

Each item is a `<div className="border border-border rounded-card overflow-hidden">` containing:

1. **Button header:**
```tsx
<button
  id={`faq-button-${index}`}
  onClick={() => toggle(index)}
  aria-expanded={openIndex === index}
  aria-controls={`faq-panel-${index}`}
  className="w-full flex items-center justify-between p-5 text-left text-text hover:bg-card transition-colors"
>
  <span className="font-bold text-sm md:text-base">{item.question}</span>
  <motion.span
    animate={{ rotate: openIndex === index ? 45 : 0 }}
    transition={{ duration: 0.2 }}
    className="text-muted text-xl flex-shrink-0 ml-4"
    aria-hidden="true"
  >
    +
  </motion.span>
</button>
```

2. **Animated answer panel:** Wrap in `<AnimatePresence initial={false}>`. Conditionally render when `openIndex === index`:
```tsx
<motion.div
  id={`faq-panel-${index}`}
  role="region"
  aria-labelledby={`faq-button-${index}`}
  initial={{ height: 0, opacity: 0 }}
  animate={{
    height: "auto",
    opacity: 1,
    transition: {
      height: { duration: 0.3, ease: [0.04, 0.62, 0.23, 0.98] },
      opacity: { duration: 0.2, delay: 0.1 },
    },
  }}
  exit={{
    height: 0,
    opacity: 0,
    transition: {
      height: { duration: 0.3, ease: [0.04, 0.62, 0.23, 0.98] },
      opacity: { duration: 0.15 },
    },
  }}
  className="overflow-hidden"
>
  <p className="px-5 pb-5 text-muted text-sm leading-relaxed">
    {item.answer}
  </p>
</motion.div>
```

Key details to NOT forget:
- `initial={false}` on `<AnimatePresence>` prevents mount animation flicker
- `overflow-hidden` on the `motion.div` prevents text overflow during height animation
- `aria-hidden="true"` on the + icon so screen readers skip it
- Custom ease `[0.04, 0.62, 0.23, 0.98]` for smooth deceleration
- Opacity staggers behind height (0.1s delay on expand, no delay on collapse) for polish
